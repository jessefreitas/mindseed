// turbo-all

1. Executar os comandos locais de build da aplicação React solicitados (ex: npm run build).
2. Executar os comandos do Git solicitados (git add, git commit, git push).
3. Executar scripts de deploy como `python deploy_vps.py`.
4. Conectar na VPS via SSH e rodar migrações, seeders e composer install quando necessário.
